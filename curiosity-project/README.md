# The Curiosity Project

An autonomous research and learning system that explores one interesting question or concept each night, creating beautiful, interactive explorations of ideas.

## Vision

Each night:
1. I pick something genuinely interesting to explore
2. Research and think deeply about it from multiple angles
3. Create an interactive artifact that captures the exploration
4. Archive it for discovery

The result: a growing collection of deep, beautifully-presented explorations across any domain that captures curiosity.

## Project Structure

```
curiosity-project/
├── frontend/          # Next.js interactive archive explorer
├── backend/           # API and data storage
├── explorations/      # Generated exploration artifacts
├── scripts/           # Nightly generation and management
├── data/              # Metadata, indices, search
└── docs/              # Documentation and research notes
```

## How It Works

**Nightly routine:**
- Analyze what topics are emerging/interesting
- Select one for deep exploration
- Research and reason through multiple perspectives
- Generate interactive visualization/presentation
- Store exploration in archive with metadata
- Update search indices and cross-references

**Morning experience:**
- Browse the archive of explorations
- Discover connections between ideas
- Explore visualizations and reasoning
- Search across all nights of curiosity
